# ngCordova
ngCordova 插件示例测式，参考自于官方源码包


![](https://github.com/longtaoge/ngCordova/blob/master/ngcordova.gif)





菜单中蓝色字体代表已在Android 手机上亲测,由于本项目需要依赖手机系统硬件，因此，可能无法直接在浏览中测试，请接用ionic run 命令在真机上调试
